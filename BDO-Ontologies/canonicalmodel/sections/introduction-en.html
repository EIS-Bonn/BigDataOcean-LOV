Introduction back to ToC

We present a vocabulary ontology called Canonical Model: Standard Variables created specifically for the Big Data Ocean project (BDO). The canonical model vocabulary represents the standards variables defined in multiple datasets. Having a semantic representation of the canonical model vocabulary allows interlinking and integrate heterogeneous data more efficiently and to benefit from the Semantic Web tools.
