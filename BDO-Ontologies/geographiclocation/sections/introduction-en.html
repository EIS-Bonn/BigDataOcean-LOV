Introduction back to ToC

We present a vocabulary ontology called Geographic Location Standards created specifically for the Big Data Ocean project (BDO). The geographic location vocabulary represents the location where was taken the data defined in multiple datasets. Having a semantic representation of the geographic location vocabulary allows interlinking and integrate heterogeneous data more efficiently and to benefit from the Semantic Web tools.
